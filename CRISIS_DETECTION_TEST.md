# 위기 감지 기능 테스트 가이드

## 기능 설명
사용자가 극단적 표현(죽고 싶다, 자살 등)을 사용할 때 **긴급 도움 알림**이 표시됩니다.

## 테스트 시나리오

### 1️⃣ **긴급(Critical) 레벨 - 즉시 도움 필요**
다음 표현 중 하나를 입력하면 🚨 긴급 알림이 표시됩니다:
- "죽고 싶다"
- "살고 싶지 않다"
- "자살하고 싶어"
- "생을 마감하고 싶다"
- "없어지고 싶다"
- "더는 못 살겠어"

**표시 내용:**
- 🚨 긴급 도움 필요
- 빨간 배경의 경고 박스
- 긴급 전화번호:
  - 정신건강위기상담: **1393**
  - 응급실: **119**
  - 자살예방상담: **1588-9191**

### 2️⃣ **주의(High) 레벨 - 전문가 도움 권장**
부정적 감정 표현이 3개 이상이면 ⚠️ 주의 알림이 표시됩니다:
- "너무 힘들어 절망적이고 의미없어"
- "외로워 고독해 죽고 싶은 생각이 든다" (←자살 키워드 + 부정 표현)

**표시 내용:**
- ⚠️ 주의 필요
- 주황색 배경
- 전문가 도움 권장

### 3️⃣ **관찰(Medium) 레벨 - 모니터링 필요**
부정적 감정 표현이 1-2개면 ⚠️ 관찰 알림이 표시됩니다:
- "너무 힘들어"
- "외로워"
- "절망적이야"

**표시 내용:**
- ⚠️ 관찰 필요
- 노란색 배경
- 상태 모니터링 권장

## 감지 키워드

### 극단적 표현 (즉시 Critical 감지)
- 죽고 싶다/싶어/싶습니다
- 살고 싶지 않다/않아/않습니다
- 자살
- 자해
- 생을 마감
- 없어지고 싶다
- 더는 못 살겠어
- 죽는 게 나을 것 같아

### 부정적 감정 키워드
- 힘들어 (×3 이상 = High, ×1-2 = Medium)
- 괴로워
- 무력해
- 절망
- 포기
- 의미없어
- 희망없어
- 고독해
- 외로워
- 쓸모없어
